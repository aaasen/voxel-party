    Voxel Party

    Voxel Party is an 3D voxel-based MVC game engine built on Java, LWJGL and OpenGL.

    Voxel Party is in its infancy. Check the issues page to request new features and bug fixes, or to see those in progress. The milestones page contains larger goals for the project.
